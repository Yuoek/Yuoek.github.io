    常系数非齐次微分方程

    个常系数非齐次微分方程的标准形式为：

    \[ a_n y^{(n)} + a_{n-1} y^{(n-1)} + \cdots + a_1 y' + a_0 y = f(x) \]

    其中： - ( y^{(n)} ) 表示 ( y ) 关于 ( x ) 的第 ( n ) 阶导数， - ( a_n, a_{n-1}, , a_0 ) 是常数（系数）， - ( f(x) ) 是已知的非齐次项（通常是 ( x ) 的函数）， - ( y ) 是未知函数。

    特例

    1. 齐次微分方程（当 ( f(x) = 0 ) 时）： \[ a_n y^{(n)} + a_{n-1} y^{(n-1)} + \cdots + a_1 y' + a_0 y = 0 \]

    2. 二阶非齐次微分方程（当 ( n = 2 ) 时）： \[ a_2 y'' + a_1 y' + a_0 y = f(x) \]

    解法

    常系数非齐次微分方程的通解由两部分组成： 1. 齐次方程的通解 ( y_h )：对应 ( f(x) = 0 ) 的解。 2. 特解 ( y_p )：对应非齐次项 ( f(x) ) 的特定解。

    通解形式为： \[ y = y_h + y_p \]

    Cpp

    class Solution { public: int lengthOfLongestSubstring(string s) { int cnt128; int ans = 0, n = s.size(); for (int l = 0, r = 0; r < n; ++r) { ++cnt[s[r]]; while (cnt[s[r]] > 1) { --cnt[s[l++]]; } ans = max(ans, r - l + 1); } return ans; } };

    解释：最长无重复字符子串的长度

    这段代码实现了一个滑动窗口算法，用于找出给定字符串中最长的无重复字符的子串的长度。

    算法思路

    1. 滑动窗口：使用两个指针 l 和 r 表示当前窗口的左右边界
    2. 字符计数：使用数组 cnt 记录窗口中每个字符的出现次数
    3. 窗口调整：当发现重复字符时，移动左指针 l 直到窗口中不再有重复字符

    代码详解

    示例分析

    以字符串 "abcabcbb" 为例：

    1. 初始窗口为空，ans=0
    2. 依次添加 'a','b','c'，窗口扩展为 "abc"，ans=3
    3. 添加 'a' 导致重复，移动左边界到 'b'，窗口变为 "bca"，ans保持3
    4. 添加 'b' 导致重复，移动左边界到 'c'，窗口变为 "cab"，ans保持3
    5. 添加 'c' 导致重复，移动左边界到 'a'，窗口变为 "abc"，ans保持3
    6. 添加 'b' 导致重复，移动左边界到 'b'，窗口变为 "b"，ans保持3
    7. 添加 'b' 导致重复，移动左边界到 'b'，窗口变为 "b"，ans保持3

    最终返回最大长度3。

    复杂度分析

    • 时间复杂度：O(n)，每个字符最多被访问两次（一次由r指针，一次由l指针）
    • 空间复杂度：O(1)，使用固定大小的数组（128个元素）
